# v0.1 Documentation Archive

These documents describe Embers v0.1 (released as `@embersjs/core@0.1.x`). They are preserved here for historical reference and to help anyone reading older code, posts, or simulation logs.

**Do not treat these as current.** They describe an architecture that was found, through five real-world Haunt simulations, to have drifted from its thesis — practices became counters with poetic names, drives could be dampened by practice (spiritual bypass as a code path), and depression/collapse couldn't happen because practices buffered all pressure. Those failure modes drove the v0.2 rebuild.

For current docs:

- `README.md` (root) — the corrected thesis and quickstart
- `docs/ARCHITECTURE.md` — v0.2 technical spec
- `docs/ROADMAP.md` — current status and what's next
- `docs/design/v0.2/foundation.md` — the design rationale for the rebuild
- `docs/authoring/` and `docs/integration/` — v0.2 authoring + integration guides

What's archived here:
